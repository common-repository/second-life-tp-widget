Adds a link to your second life location via slurl. It makes it easy to add a location based on Second Life's coordinate system.


== Description ==

Requirements:
* Adds a method of inputting your Second Life coordinates into the widget, then generates a slurl in your sidebar
* Allows for the user to change the heading of the widget
* Opens slurl in a new window
* Hides itself if the widget is empty


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `sl-tp-widget.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use the 'Apperance' menu to place the widget into your blog
1. Enter the relevant information in the widget editor

== Frequently Asked Questions ==

= Why use this wiget? =

In case you ever decide to change locations, there is no need to start changing links site wide.  Just change the coordinates in one place and you're good to go!

It's a pretty basic and simple to use plugin.
